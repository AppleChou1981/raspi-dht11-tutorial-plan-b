DHT11 溫溼度數據收集：樹莓派本地中繼與直連 LINE Bot 方案 (方案 B)

⬅️ 返回【方案 A：GAS 快速部署方案】首頁
⚠️ 重要提醒：請先插妥杜邦線並確認所有接線正確後再行送電！若是電源（VCC）與接地（GND）接反，會導致硬體晶片過熱燒毀！
💡 架構導讀（方案 B）： 本方案由樹莓派作為本地中控發起核心。讀取 DHT11 後，樹莓派將引入 GCP 安全 JSON 憑證，直接透過 Google Sheets API V4 寫入指定的試算表 ID，並平行呼叫 LINE 官方伺服器 API 發送訊息。此方案傳輸路徑最直接，數據自主權最高。
第一步：取得 LINE 通訊憑證 (Token 與 ID)
1. 獲取 Your user ID：
登入 LINE Developers ➡️ 點進您的 Channel ➡️ Basic settings 頁籤 ➡️ 滑至最底部複製 Your user ID (U開頭亂碼)。
2. 獲取 Access Token：
切換至 Messaging API 頁籤 ➡️ 滑至最底部 ➡️ 在 Channel access token 欄位點擊「Issue」並複製該超長亂碼金鑰。

⚠️ 關鍵動作：請務必掃描頁面上的 QR Code 將機器人**加入好友**，否則系統無權限推送訊息。

第二步：Google Cloud Platform (GCP) JSON 憑證申請

樹莓派必須透過此憑證才能安全地直連 Google 試算表：

1. 啟用 API： 前往 GCP 控制台，建立專案並啟用 Google Sheets API。
2. 建立服務帳戶： 進入「憑證」➡️「建立憑證」➡️「服務帳戶」。建立後，複製該帳戶的 Email 地址。
3. 下載 JSON 金鑰： 點擊剛建立的服務帳戶 ➡️「金鑰」頁籤 ➡️「新增金鑰」➡️「建立新金鑰」➡️ 選擇 JSON。下載後請將檔案命名為 credentials.json 並放在樹莓派程式資料夾。
4. 共用試算表： 開啟您的 Google 試算表，點擊右上角「共用」，將上述**服務帳戶 Email** 加入並給予「編輯者」權限。

觀察瀏覽器上方網址列，結構如下圖。請僅複製兩條斜線中間的那一長串亂碼：

https://docs.google.com/spreadsheets/d/【 只要複製這整串隨機亂碼，這就是試算表 ID 】/edit#gid=0
第四步：硬體腳位與型號提醒

⚠️ 型號與電阻關鍵確認：

  • 3 腳位模組（常見藍色小板）： 通常已內建上拉電阻，不需外接，接 VCC、GND、DATA 即可。
  • 4 腳位元件（裸件）： 內部無整合電阻，必須在 VCC 與 DATA 之間手動並聯一個 4.7kΩ ~ 10kΩ 的上拉電阻，否則訊號會因浮動導致讀取失敗。

第五步：樹莓派本地中繼與直連 LINE Bot 控制主程式

此程式碼將同時調用 google-api-python-client (須先安裝) 與 LINE API：

import time
import requests

# 記得先 pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib

# ==================== 1. 參數與憑證設定 ====================

TARGET_USER_ID = "貼上你的Your_user_ID"
JSON_KEY_FILE = "credentials.json"  # 從 GCP 下載的金鑰檔案

dhtDevice = adafruit_dht.DHT11(board.D4)

def send_to_line_bot_direct(temp, humi):
    """樹莓派直接呼叫 LINE 官方主機 API 推播"""
    line_api_url = "https://api.line.me/v2/bot/message/push"
    current_time = time.strftime("%Y-%m-%d %H:%M:%S", time.localtime())
    message_text = f"\n🌡️ 樹莓派溫溼度直連回報\n-------------------\n⏰ 時間：{current_time}\n°C 溫度：{temp} °C\n💧 濕度：{humi} %"
    
    headers = {
        "Content-Type": "application/json",
        "Authorization": f"Bearer {LINE_ACCESS_TOKEN}"
    }
    payload = {
        "to": TARGET_USER_ID,
        "messages": [{"type": "text", "text": message_text}]
    }
    try:
        response = requests.post(line_api_url, json=payload, headers=headers, timeout=10)
        if response.status_code == 200:
            print("【LINE Bot】手機推播成功！")
    except Exception as e:
        print(f"【LINE Bot】通訊異常: {e}")
